Question-1: What are the different ways to select an element in the DOM?

5 ways to select an element in the DOM:

  1. getElementById()
  2. getElementsByClassName()
  3. querySelector()
  4. querySelectorAll()
  5. getElementsByTagName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. innerHTML: Inner HTML refers to the HTML markup within a specific element.
  2. innerText: Inner text only includes the visible text content of an element.
  3. textContent: TextContent property retrieves all the text within an element, including any hidden text.

Question-3: What is event delegation in the DOM?

Use a single event listener for multiple elements, ensuring it functions even when new elements are added. It bubbles up through each parent in the DOM hierarchy.

Question-4: What is event bubbling in the DOM?

Event bubbling is from the child to the parent. like down to up towards. Event bubbling is a default behavior. Event bubbling opposite of event capturing.

Question-5: How do you create, add, and remove elements using JavaScript?

  1. create elements: document.createElement()
  2. add elements: parent element.appendChild(childElement)
  3. remove elements: parent element.removeChild(childElement)
